Trabalho feito para as disciplinas: Engenharia de Software 1 e Banco de Dados 1.

Professores: Rogerio Eduardo Garcia, Ronaldo Celso Messias Correia.

Github do trabalho: https://github.com/vinico-o/Sistema-SAF

O sistema tem como objetivo ser auxiliar dos donos e funcionários de clubes de futebol, permitindo cadastrar clubes, usuários, receitas e despesas, jogadores, partidas, visualizar auditorias e gerar relatórios. Todas as informações são salvas em banco de dados SQLITE.
Para melhor entendimento do projeto, é recomendado a leitura do Documento de Requisitos na Pasta "Documentos".

Para executar, compile e rode "TelaPrincipal.java".
